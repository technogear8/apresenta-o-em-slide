Guia de Instalação

  1. Baixe a ISO oficial.
  2. Use o BalenaEtcher para criar o pendrive.
  3. Dê boot pelo USB (F12/F10).
  4. Siga o instalador gráfico.

Domine o Terminal

O terminal não é um bicho de sete cabeças. É a sua ferramenta mais rápida.
